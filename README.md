# Essential Redux Hosting

Hosted configuration page for the Essential Redux Pebble watchface.

## GitHub Pages

Use this repo with GitHub Pages enabled from the `main` branch root.

Expected config URL:

```text
https://lyle-morris.github.io/Hosting/
```

## Files

- `index.html` — Pebble configuration page baseline
- `.nojekyll` — keeps GitHub Pages from applying Jekyll processing

## Notes

This setup follows the same separate-hosting repo pattern used by DayPal / DayPal-Hosting.
